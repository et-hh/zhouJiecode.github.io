    # 括号之前一定要加分号

    var b, a = []
(b || []).map(() => {})

    这种会报错 Uncaught TypeError: [] is not a function，括号前必须有分号，否则浏览器会认为括号前的内容是函数

    # async await

    使用async await来优化代码结构

    async function fun(){
    var data = await getData()
    ...
}

    当顺序多个异步调用时，可以使用async await，也可以直接使用promise 当每个异步后逻辑较长时，应使用promise而非aysnc await，这样每个异步及其后续逻辑放在一个独立then函数中，结构更清晰

    # 用户信息及token应保存于程序内部

    用户信息及token应保存于程序内部，只有刷新时才于main.js去sessionStorage获取，后续项目代码中应直接使用程序内部变量，不能再使用sessionStorage获取信息（页面双开时使用localstorage可能会发生冲突，sessionstorage不会，因为sessionstorage只作用于当个标签页）

    # 在router.beforeEach钩子中进行鉴权处理

    router.beforeEach（(to, from, next) => {
    // 登录鉴权失败
    if(!hasLoginRight(to)){
        if(from.path !== '/'){
            //401 timeout
            next()
            return
        }
        goToLogin(next)
    }

    // 菜单鉴权失败
    if(!hasMenuRight(to)){
        if(hasMenuRight(from) && from.path !== '/' && to.path !== '/'){
            //保持路由
            next({
                path: from.path
            });
            return;
        }
        goToLogin(next)
        return;
    }

    next()
}）
